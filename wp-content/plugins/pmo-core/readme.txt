== Description ==

PMO Core plugin for Lagos State Project Monitoring Office Digital Portal. Provides custom post types, taxonomies, and functionality for projects, news, events, publications, tenders, and leadership management.

== Features ==

* Custom post types for Projects, News, Events, Publications, Tenders, and Leadership
* Custom taxonomies for organizing content (Project Categories, LGAs, Project Status, etc.)
* Pre-configured user roles for PMO staff
* REST API support for all custom post types
* Admin interfaces for managing all content

== Installation ==

1. Upload the plugin to wp-content/plugins/
2. Activate the plugin from WordPress admin
3. The custom post types and taxonomies will be automatically registered
