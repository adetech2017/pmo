== Description ==

Modern, responsive WordPress theme for Lagos State Project Monitoring Office Digital Portal. Features government branding, accessibility (WCAG 2.1 AA), and clean, professional design.

== Features ==

* Fully responsive design
* WCAG 2.1 AA accessibility compliance
* Custom logo support
* Multiple navigation menus
* Customizable colors and branding
* Clean and modern design
* Search-engine optimized
* Fast loading performance

== Installation ==

1. Upload the theme to wp-content/themes/
2. Activate the theme from WordPress admin
3. Customize the theme colors and settings in the WordPress Customizer
